# Rental Expense Forecaster

A React-based financial simulation tool for landlords and property managers.

## Documentation

For comprehensive documentation, please refer to [GEMINI.md](./GEMINI.md).

## Getting Started

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Run the development server:
    ```bash
    npm run dev
    ```

3.  Build for production:
    ```bash
    npm run build
    ```
